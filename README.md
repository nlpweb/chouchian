Chouchian (抽籤)
================

A Lottery based on `Javascript` / `Google Spreadsheet`


Local Dev
=========

```
python server -d -p 5000
```

Features
========

* Modify the google spreadsheet to change the `data content` (e.g., student name)
* Use hashtag to change `data class` (e.g., class name)
